## ✨ Features

* 🖼️ Displays services and portfolio sections
* 🎨 Clean layout using HTML & CSS
* 🖥️ Desktop view design
* 📄 Multiple sections: Home, About, Services, Portfolio, Contact
* ⚡ Basic hover effects on buttons and cards
* 📌 Simple typography and color scheme

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Hazem-2002/digital-marketing-website.git
```

2. Navigate to the project folder:

```bash
cd digital-marketing-website
```

3. Open the project:

Open `index.html` in your browser
